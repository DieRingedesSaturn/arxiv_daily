**核心结论**: 研究发现外部四极矩畸变在轨道动力学和吸积形态上留下了显著印记。辐射区外缘的分布与辐射压及气体压主导地位的转换点紧密耦合，证明了局部时空几何不仅是数学抽象，更会对吸积流的形态和辐射观测特征产生可探测的影响。这为理解非孤立致密天体的强场物理过程提供了重要理论支撑。
