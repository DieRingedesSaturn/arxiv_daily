# Source: 1A_1118-61

- **类别**: NSXRB
- **简介**: Be X-ray binary (BeXRB) containing a neutron star.

### ATel 17667 | 11 Feb 2026 UT
**设备**: XRISM

1A 1118-61 是一颗典型的 Be/X 射线双星（NSXRB），近期处于爆发活动期。XRISM 卫星对其进行了高分辨率 X 射线光谱观测，旨在探究吸积流与中子星磁场相互作用的物理机制。

**建议：** 由于该源属于 NSXRB，与贵组核心研究（BHXRB/AGN）相关性较低，且 1A 1118-61 爆发时的光学亮度可能较暗，建议优先关注爆发期间的长期光变监测。若爆发期间光学亮度达到 16 等以上，可利用 1m 望远镜获取低分辨率光谱，监测 Hα 发射线强度变化，以分析 Be 星吸积盘的演化及轨道周期调制。
