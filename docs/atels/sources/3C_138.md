### ATel 17681: [Fermi-LAT detection of renewed gamma-ray activity from the CSS 3C 138](https://www.astronomerstelegram.org/?read=17681)
- **日期**: 14 Feb 2026 UT

**爆发速递**: Fermi-LAT 探测到致密陡谱类星体 3C 138 的伽马射线活动增强。2026年2月11日，其日均伽马射线通量较 4FGL-DR4 平均水平提升超过 100 倍，光子指数为 2.0。**观测短评**: 该源为已知活跃的 AGN，此次爆发幅度显著，具有极高的多波段联合观测价值。鉴于其作为类星体的性质，建议利用 Swift 或其他空间设备进行 X 射线监测，并结合光学望远镜开展测光或光谱观测以捕捉可能的吸积盘或喷流变化。由于该源亮度及性质，适合纳入多波段协同观测计划。
