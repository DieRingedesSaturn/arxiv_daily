### ATel 17681: [Fermi-LAT detection of renewed gamma-ray activity from the CSS 3C 138](https://www.astronomerstelegram.org/?read=17681)
- **日期**: 14 Feb 2026 UT

**爆发速递**: Fermi-LAT 探测到紧凑陡谱源 (CSS) 3C 138 的伽马射线活动再次增强。该源表现出显著的伽马射线流量提升，预示着其喷流活动进入活跃期。**观测短评**: 3C 138 作为典型的 CSS 源，其伽马射线爆发对于研究年轻 AGN 的喷流物理具有重要意义。该源光学亮度通常在 18-19 等左右，处于 1m 望远镜测光极限内，可尝试进行多色测光监测。建议结合 Swift 或 XMM-Newton 进行多波段能谱分析，以研究吸积与喷流的关联。
