# Source: A_1118-616

- **类别**: NSXRB
- **简介**: Be/X-ray binary (BeXRB) containing a neutron star.

### ATel 17690 | 19 Feb 2026 UT
**设备**: Fermi/GBM

核心现象：Fermi/GBM 探测到 A 1118-616 的 X 射线爆发。该天体是一颗由 Be 星和中子星组成的 Be/X 射线双星。对于 1m 望远镜，由于该源处于爆发状态，其光学对应体可能变亮，建议监测其光学波段的光变曲线以寻找轨道调制或吸积盘演化特征；若能进行低分辨率光谱观测，可监测其 H-alpha 发射线轮廓的变化，这对研究 Be 星的星周盘结构及吸积物理有重要意义。但由于其主要是 NSXRB，与贵组主要研究领域关联度有限，优先级建议较低。
