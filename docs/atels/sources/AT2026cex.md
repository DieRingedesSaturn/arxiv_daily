# Source: AT2026cex

- **类别**: Other
- **简介**: LFBOT (发光快速蓝光学瞬变源)

### ATel 17677 | 12 Feb 2026 UT
**设备**: FAST

该简报报道了对LFBOT候选体AT2026cex进行的射电波段观测，主要利用FAST望远镜在L波段搜索是否存在快速单脉冲信号。LFBOT作为一种极高亮度的瞬变源，其高能辐射机制与吸积过程密切相关。虽然该源处于LFBOT范畴，但鉴于其距离较远且主要涉及射电观测，1m光学望远镜难以进行有效的测光或光谱后随（通常星等较暗，超出了16等光谱极限）。建议保持关注该类源的早期多波段触发，若后续发现有处于宿主星系内的、符合光学观测极限的近邻LFBOT，可利用1m望远镜进行长期的光变监测以研究其吸积演化。
