# Source: BZBJ0958+6533

- **类别**: AGN
- **简介**: 耀变体（Blazar），属于活动星系核的一种。

### ATel 17672 | 12 Feb 2026 UT
**设备**: Swift-XRT, Swift-UVOT, Kanata telescope, STELLA

BZBJ0958+6533 近期表现出显著的增亮趋势，Swift 多波段观测显示其光谱能量分布（SED）发生演变，通常对应于喷流活动的增强。该源目前仍处于活跃期，是研究 AGN 喷流物理的理想目标。利用 1m 望远镜可进行长期的光学多波段测光监测，通过绘制光变曲线分析其短时标光变特征（如闪烁），并与 X 射线数据比对研究其辐射机制。
